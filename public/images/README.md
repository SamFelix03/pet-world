# Pet Fallback Images

Place static fallback images here that will be shown if Lottie animations fail to load.

## Recommended Files

- `stage-0-egg.png` - Static image for Egg stage
- `stage-1-baby.png` - Static image for Baby stage
- `stage-2-teen.png` - Static image for Teen stage
- `stage-3-adult.png` - Static image for Adult stage

## Image Specifications

- **Format:** PNG (with transparency) or SVG
- **Size:** Recommended 512x512px or larger (will be scaled)
- **Background:** Transparent preferred
- **Aspect Ratio:** Square (1:1)

These images serve as fallbacks and should represent the same pet stages as your animations.
